# AAA-App

Реализация системы AAA.

# Build & Run & Test application
В каждом пути задания имеются скрипты сборки, запуска и тестирования.

## Build
Запустить скрипт сборки — `BUILD.sh`
```bash
./BUILD.sh
```


## Run
Запустить приложение `RUN.sh`
```bash
./RUN.sh <Параметры>
```

## Test
Запустить скрипт `TEST.sh`
```bash
./TEST.sh
```
